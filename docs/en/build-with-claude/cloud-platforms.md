# Claude on Cloud Platforms

Use Claude via AWS Bedrock. Supports most models.

## Amazon Bedrock (Legacy)
Legacy Bedrock integration with limited model support.

## Claude Platform on AWS
Full Claude Platform available on AWS with enhanced features.

## Google Cloud (Vertex AI)
Use Claude via Google Cloud Vertex AI.

## Microsoft Foundry (Azure)
Use Claude via Microsoft Foundry on Azure.
