# Управление контекстом

## Контекстные окна
До 1 млн токенов для больших документов, кодовых баз и длинных диалогов.

## Компактификация
Уменьшение размера контекста с сохранением ключевой информации.

## Редактирование контекста
Изменение контекста во время длинных диалогов.

## Кэширование промптов
Кэширование промптов для снижения затрат (экономия 50% на кэшированном вводе).

## Системные сообщения в середине диалога
Внедрение системных сообщений в середине диалога.

## Построение режма оркестрации
Построение сложных шаблонов оркестрации.

## Диагностика кэша (Бета)
Мониторинг производительности кэша.

## Подсчёт токенов
Подсчёт токенов перед отправкой запросов.
