# Работа с файлами (Working with Files)

## Files API

Загрузка и управление файлами для Claude.

## Поддержка PDF

Обработка текстового и визуального контента из PDF. Поддерживает:
- PDF, закодированные в Base64
- URL-адреса
- Идентификаторы файлов Files API

## Изображения и зрение

Обработка и анализ визуальных входных данных. Поддерживает:
- Изображения, закодированные в Base64
- URL-адреса
- Несколько изображений на сообщение
