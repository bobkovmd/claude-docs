# Причины остановки и резервирование (Stop Reasons and Fallback)

Ответы Claude API включают поле `stop_reason`, указывающее, почему модель прекратила генерацию.

## Причины остановки
| Значение | Значение |
|---|---|
| `end_turn` | Естественное завершение |
| `max_tokens` | Достигнут лимит токенов |
| `stop_sequence` | Достигнута последовательность остановки |
| `tool_use` | Запрошен вызов инструмента |
| `refusal` | Классификатор безопасности отклонил |
| `pause_turn` | Достигнут лимит итерации серверного инструмента |

# Резервный кредит (Fallback Credit)

Избежание двойной оплаты кэша промптов при повторных попытках отклонённых запросов.

## Как это работает
1. Активируйте с помощью `anthropic-beta: fallback-credit-2026-06-01`
2. Прочитайте `fallback_credit_token` из `stop_details` отказа
3. Отправьте токен при повторной попытке
4. Повторная попытка тарифицируется, как если бы диалог всегда был на новой модели
