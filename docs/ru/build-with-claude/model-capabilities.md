# Возможности модели (Model Capabilities)

## Расширенное мышление
Улучшенное рассуждение с прозрачным пошаговым процессом мышления.

### Как это работает
API-ответ включает блоки контента `thinking`, за которыми следуют блоки контента `text`.

### Ключевые параметры
- `budget_tokens`: Максимум токенов для внутреннего рассуждения (мин. 1,024)
- `display`: "summarized" или "omitted"

### Потоковое мышление
Потоковая передача через SSE. Контент мышления поступает через события `thinking_delta`.

## Адаптивное мышление (Claude Opus 4.8+)
- По умолчанию мышление ВЫКЛЮЧЕНО
- Глубина контролируется через параметр `effort`
- При `max`/`xhigh` effort установите большой максимальный выход (начните с 64k токенов)

### Уровни Effort
| Уровень | Лучше всего для |
|---|---|
| `max` | Задачи, требующие высокого интеллекта |
| `xhigh` | Лучший выбор для программирования и агентных сценариев |
| `high` | Минимум для чувствительных к интеллекту сценариев |
| `medium` | Чувствительные к стоимости |
| `low` | Чувствительные к задержке |

## Бюджеты задач (Beta)
Рекомендательный бюджет токенов для полного агентного цикла. Саморегулирование расхода токенов.

## Быстрый режим (Research Preview)
До 2.5x больше выходных токенов в секунду. $50/MTокен за вывод для Opus 4.8.

## Структурированные выходы
Гарантированное соответствие схеме через JSON-выходы или строгое использование инструментов.

## Цитирование
Обоснование ответов в исходных документах с точными ссылками на предложения/фрагменты.

## Потоковая передача сообщений
Инкрементальные ответы через SSE. Поддерживает текст, использование инструментов и дельты мышления.

## Пакетная обработка
Асинхронная обработка со снижением стоимости на 50%. До 100K запросов в пакете.

## Результаты поиска
Естественные ссылки для RAG с указанием источника.

## Многоязычная поддержка
Сильная кросс-лингвистическая производительность относительно английского (100% базовый уровень).

## Эмбеддинги
Anthropic не предлагает собственную модель эмбеддингов. Рекомендуется Voyage AI.
