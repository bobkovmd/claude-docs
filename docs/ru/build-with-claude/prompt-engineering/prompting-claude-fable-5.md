# Промптинг Claude Fable 5

## Обзор

Это руководство охватывает паттерны промптинга и скаффолдинга, специфичные для **Claude Fable 5** и **Claude Mythos 5**. Claude Fable 5 нацелен на задачи, которые ранее были слишком сложными, длительными или неоднозначными для предыдущих моделей — особенно эффективен для сквозной работы, занимающей часы, дни или недели.

## Ключевые поведенческие отличия от Claude Opus 4.8

### Улучшения возможностей
- **Автономность на дальние горизонты** — поддерживает продуктивный вывод в течение нескольких дней при целенаправленных запусках
- **Правильность с первого раза** — однократная реализация систем, которые ранее требовали дней итераций
- **Зрение** — интерпретирует плотные технические изображения, веб-приложения и детальные скриншоты с более высокой точностью
- **Корпоративные рабочие процессы** — профессиональный вывод для финансового анализа, электронных таблиц, презентаций, документов
- **Обзор и отладка кода** — заметно более высокая полнота обнаружения ошибок
- **Навигация в неоднозначности** — хорошо справляется со сложными, многопоточными запросами
- **Делегирование и сотрудничество** — значительно более надёжная отправка параллельных субагентов

### Классификаторы безопасности

Claude Fable 5 запускает классификаторы безопасности, нацеленные на:
- Оффенсивные кибербезопасные техники (эксплойты, вредоносное ПО, инструменты атак)
- Контент по биологии и наукам о жизни (лабораторные методы, молекулярные механизмы)
- Извлечение обобщённого мышления модели

## Паттерны промптинга

### 1. Более длинные ходы по умолчанию
Отдельные запросы по сложным задачам могут выполняться **много минут** при более высоких настройках effort; автономные запуски могут длиться **часами**.

### 2. Рассмотрите все уровни Effort
- Используйте **`high`** по умолчанию для большинства задач
- Используйте **`xhigh`** для большинства чувствительных к возможностям рабочих нагрузок
- Используйте **`medium`** или **`low`** для рутинной работы

### 3. Сильное следование инструкциям
Краткие инструкции теперь эффективно направляют большинство поведений — нет необходимости перечислять каждое поведение по имени.

### 4. Обоснование утверждений о прогрессе во время длинных запусков
Проинструктируйте Claude Fable 5 проверять прогресс по сравнению с реальными результатами инструментов. В тестировании Anthropic это **практически устранило сфабрикованные отчёты о статусе**.
