# Инструменты промптинга в Claude Console

## Обзор

Claude Console предоставляет интегрированный набор инструментов для создания, уточнения и развёртывания промптов. Типичный рабочий процесс состоит из трёх этапов: **сгенерировать → шаблонизировать → улучшить**.

## 1. Генератор промптов

- **Назначение:** Решает «проблему пустой страницы», направляя Claude к созданию высококачественных шаблонов промптов для конкретных задач, следуя лучшим практикам Anthropic.
- **Совместимость:** Работает со всеми моделями Claude, включая модели с расширенным мышлением.
- **Ресурсы:** Доступен непосредственно в панели управления Console.

## 2. Шаблоны и переменные промптов

### Основная концепция

API-вызовы комбинируют два типа контента:
- **Фиксированный контент** — Статические инструкции/контекст, постоянные между взаимодействиями
- **Переменный контент** — Динамические элементы (пользовательский ввод, контент RAG, история диалогов, результаты инструментов)

### Синтаксис

Заглушки используют **двойные скобки**: `{{variable_name}}`

### Ключевые преимущества
- **Согласованность** — единообразная структура промпта между вызовами
- **Эффективность** — меняйте переменные без переписывания промптов
- **Тестируемость** — быстро тестируйте граничные случаи
- **Масштабируемость** — упрощает управление промптами
- **Контроль версий** — отслеживайте изменения отдельно от динамических входных данных

> **Примечание:** `claude.ai` в настоящее время **не** поддерживает шаблоны и переменные промптов.

## 3. Улучшатель промптов

### Назначение
Автоматизированный анализ и улучшение промптов, оптимизированных для **сложных задач, требующих высокой точности**.

### Как это работает (4-шаговый процесс)

1. **Идентификация примеров** — Находит и извлекает примеры из шаблона
2. **Первый черновик** — Создаёт структурированный шаблон с чёткими разделами и XML-тегами
3. **Уточнение цепочки рассуждений** — Добавляет подробные инструкции по рассуждению
4. **Улучшение примеров** — Обновляет примеры для демонстрации нового процесса рассуждения

### Особенности вывода
- Подробные инструкции по цепочке рассуждений
- Чёткая организация с XML-тегами
- Стандартизированное форматирование примеров с пошаговым рассуждением
- Стратегические предзаполнения, направляющие начальные ответы

### Генератор тестовых случаев
Сгенерируйте примеры входных данных → Получите ответы Claude → Отредактируйте для соответствия идеальным выводам → Добавьте отполированные примеры
