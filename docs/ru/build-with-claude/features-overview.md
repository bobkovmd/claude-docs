# Обзор возможностей Claude Platform

## Организация API-поверхности

API Claude организован в **пять основных областей**:

1. **Возможности модели** — Контроль рассуждений и форматирования ответов
2. **Инструменты** — Позволяют Claude выполнять действия в вебе или в вашей среде
3. **Инфраструктура инструментов** — Обнаружение и оркестрация в масштабе
4. **Управление контекстом** — Поддержание эффективности длительных сессий
5. **Файлы и ресурсы** — Управление документами и данными, передаваемыми Claude

## Классификации доступности функций

| Классификация | Ключевые детали |
|---|---|
| **Beta** | Предварительные функции; могут измениться/быть отключены |
| **Generally Available (GA)** | Стабильные, готовые к промышленному использованию |
| **Deprecated** | Работающие, но больше не рекомендуемые |
| **Retired** | Больше не доступные |

## Возможности модели

| Функция | Описание | Доступность |
|---|---|---|
| Контекстные окна | До 1 млн токенов | Все платформы |
| Адаптивное мышление | Динамический контроль глубины мышления | Все платформы |
| Пакетная обработка | Асинхронная массовая обработка со скидкой 50% | Claude API, AWS |
| Цитирование | Обоснование ответов в исходных документах | Все платформы |
| Effort | Контроль использования токенов vs. полнота ответа | Все платформы |
| Расширенное мышление | Прозрачность пошагового рассуждения | Все платформы |
| Поддержка PDF | Обработка текстового и визуального контента из PDF | Все платформы |
| Структурированные выходы | Гарантированное соответствие схеме | Все платформы |

## Инструменты

### Серверные инструменты
| Функция | Описание |
|---|---|
| Выполнение кода | Изолированное выполнение кода; бесплатно с web search/fetch |
| Веб-выборка (Web fetch) | Получение полного контента с веб-страниц и PDF |
| Веб-поиск (Web search) | Дополнение Claude актуальными данными из веба |

### Клиентские инструменты
| Функция | Описание |
|---|---|
| Bash | Выполнение команд и скриптов оболочки |
| Computer use | Управление интерфейсами через скриншоты, мышь, клавиатура |
| Память | Сохранение/извлечение информации между диалогами |
| Текстовый редактор | Создание и редактирование текстовых файлов |
