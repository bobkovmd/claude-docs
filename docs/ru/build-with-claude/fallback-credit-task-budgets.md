# Резервный кредит (Fallback Credit)

Избежание двойной оплаты затрат на кэш промптов при повторных попытках запросов, отклонённых другой моделью.

## Порядок действий
1. Активируйте с помощью `anthropic-beta: fallback-credit-2026-06-01`
2. Прочитайте `fallback_credit_token` и `fallback_has_prefill_claim` из отказа
3. Сформируйте повторный запрос с токеном
4. Отправьте повторный запрос с тем же заголовком beta

## Ограничение
Токен нельзя использовать из результатов Message Batches — только напрямую через Messages API.

# Бюджеты задач (Beta)

Рекомендательный бюджет токенов для полного агентного цикла.

## Использование
```json
"output_config": {
    "effort": "high",
    "task_budget": {"type": "tokens", "total": 64000}
}
```

## Ключевые моменты
- Рекомендательный, а не принудительный (мягкая подсказка)
- Минимум: 20,000 токенов
- Дополняет параметр effort
- Виден только модели (не в API-ответе)

# Быстрый режим (Research Preview)

До 2.5x больше выходных токенов в секунду по премиальному тарифу.

## Ценообразование
- Ввод: $10/MTокен
- Вывод: $50/MTокен

## Требования
- Доступ к Research preview
- Только Claude Opus 4.8
- Заголовок beta: `fast-mode-2026-02-01`
