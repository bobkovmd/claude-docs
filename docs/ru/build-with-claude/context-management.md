# Контекстные окна (Context Windows)

До 1 млн токенов для больших документов, кодовых баз, длинных диалогов.

# Компактификация (Compaction)

Уменьшение размера контекста с сохранением ключевой информации.

# Редактирование контекста (Context Editing)

Изменение контекста во время длинных диалогов.

# Кэширование промптов (Prompt Caching)

Кэширование промптов для снижения затрат (экономия 50% на кэшированном вводе). Размещайте `cache_control` в конце блоков контента.

# Системные сообщения в середине диалога (Mid-conversation System Messages)

Внедрение системных сообщений в середине диалога.

# Построение режима оркестрации (Build an Orchestration Mode)

Построение сложных паттернов оркестрации с несколькими агентами.

# Диагностика кэша (Beta)

Мониторинг производительности кэша и частоты попаданий.

# Подсчёт токенов (Token Counting)

Подсчёт токенов перед отправкой запросов для оценки стоимости.
