# Claude на облачных платформах

## Amazon Bedrock
Используйте Claude через AWS Bedrock. Поддерживает большинство моделей.

## Amazon Bedrock (Устаревшая)
Устаревшая интеграция Bedrock с ограниченной поддержкой моделей.

## Claude Platform на AWS
Полная Claude Platform доступна на AWS с расширенными возможностями.

## Google Cloud (Vertex AI)
Используйте Claude через Google Cloud Vertex AI.

## Microsoft Foundry (Azure)
Используйте Claude через Microsoft Foundry на Azure.
