# Пакетная обработка (Batch Processing)

Асинхронная обработка больших объёмов запросов со снижением стоимости на 50%.

## Ограничения
| Ограничение | Значение |
|---|---|
| Максимум запросов в пакете | 100,000 ИЛИ 256 МБ |
| Время обработки | Большинство в течение 1 часа; максимум 24 часа |
| Доступность результатов | 29 дней |

## Ценообразование (50% от стандартного)
| Модель | Ввод (пакет) | Вывод (пакет) |
|---|---|---|
| Claude Opus 4.8 | $2.50/MTокен | $12.50/MTокен |
| Claude Sonnet 4.6 | $1.50/MTокен | $7.50/MTокен |
| Claude Haiku 4.5 | $0.50/MTокен | $2.50/MTокен |

## Результаты поиска (Search Results)

Естественные ссылки для RAG с указанием источника. Правило «всё или ничего» для цитирования.
