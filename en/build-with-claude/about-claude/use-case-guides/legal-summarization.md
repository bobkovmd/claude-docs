# Legal summarization with Claude

A comprehensive guide to using Claude's NLP capabilities for summarizing legal documents, extracting key information, and expediting legal research.

## When to Use Claude for Legal Summarization

- High-volume document review
- Automated metadata extraction
- Standardized summaries
- Precise citations
- Streamlined legal research

## Implementation

- Model: Opus 4.8 for high accuracy, Haiku 4.5 for cost-effective large volumes
- Extract and clean text from PDFs using pypdf
- Create structured prompts with XML-tagged sections
- Evaluate with ROUGE scores and human review
